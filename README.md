# Food Ordering System
A web-based platform designed to help restaurants and customers manage food orders, menus, and delivery in one centralized system. The system improves order handling, customer feedback, and efficiency in the food ordering process. Customers can browse the menu, place orders with delivery details and remarks, and leave ratings and reviews, while admins can manage the menu, accept or reject orders, update order status, and print receipts.

## Features

- User registration and secure login (role-based access for Admin and Customer)
- Menu management(add, edit, delete, and view food items by category: Fast Food, Desserts, Drinks)
- Shopping cart with optional remark per item (“No ice”, “Less sugar”)
- Order placement with delivery address and phone; checkout modal and order confirmation page
- Real-time incoming orders on admin dashboard 
- Accept or reject orders (admin); then update status: Preparing, Out for delivery, Delivered
- Customer order history (My Orders) with order status and view order detail
- Order confirmation page/notification after placing order (with status: “Waiting for restaurant to accept”)
- Rating and review system (1–5 stars and optional comment)
- Print receipt for admin (print-friendly receipt view)
- Contact support page and user profile management
- Centralized dashboards for customer (menu + cart) and admin (menu + orders)
